1. Estructura Básica de un documento HTML

Todo documento HTML comienza con una estructura básica. Veamos cómo se compone: 


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
</head>
<body>
    <!-- Contenido va aquí -->
</body>
</html>
  • < !DOCTYPE HTML>: Declara que el documento es HTML 5.
  • <html lang="es">: Indica el idioma del documento
  • <head>: Contiene información meta sobre el documento
  • <body>Contiene el contenido visible de la página

2. Etiquetas Fundamentales

<p> - Párrafos

La etiqueta <p> se utiliza para definir un parrafo de texto

<p> Este es un párrafo de ejemplo.</p>

<h1> a <h6> - Encabezados

Los encabezados van del nivel 1 al 6, siendo <h1> el más importante.


<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Encabezado Nivel 3</h3>

3. Formato y Estilo de Texto

Negrita y Cursiva

Utiliza <strong> para negrita y <em> para cursiva


<p>Esto es <strong>negrita</strong> y esto es <em>cursiva</em>.</p>

Citas y Código

Para citas en línea usa <q> y para bloques de codigo <code> o <pre>.


<p>Ella dijo: <q>La práctica hace al maestro.</q></p>
<pre><code>
function saludo() {
    console.log('Hola Mundo');
}
</code></pre>

4. Enlaces e Imágenes

Enlaces (<a>)

Crea hipervinculos a otras paginas o recursos.

<a href="https://www.ejemplo.com">Visita Ejemplo</a>

Imágenes (<img>)

Incorpora imágenes en tu página

<img src="ruta/image.jpg" alt="Descripcion de la imagen">

5. Listas y Tablas

Listas Ordenadas y Desordenadas

Utiliza <ul> y <ol> Para crear listas.


 <ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
</ul>
                        
<ol>
    <li>Paso 1</li>
    <li>Paso 2</li>
</ol>

Tablas

Crea tablas para mostrar datos estructurados


<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Ana</td>
        <td>25</td>
    </tr>
</table>

6.Formularios y Entradas de Usuario

Elementos Básicos de un Formulario

Los Formularios permiten recopilar información del usuario


<form action="procesar.php" method="POST">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
                            
    <label for="email">Correo Electrónico:</label>
    <input type="email" id="email" name="email" required>
                            
    <button type="submit">Enviar</button>
</form>

Las etiquetas semánticas ayudan a definir la estructura y el significado del contenido:

  • <header>: Encabezado de la pagina o sección
  • <nav>: Sección de navegación
  • <section>: Sección genérica del contenido.
  • <article>Contenido independiente
  • <aside>: Contenido relacion pero no central.
  • <footer>:Pie de página

Usar estas etiquetas mejora la accesibilidad y el SEO de tu sitio.
