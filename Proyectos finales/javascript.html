1. ¿Que es JavaScript

JavaScript es un lenguaje de programacion interpretado, orientado a objetos y utilizado principalmente para desarrollar paginas webs interactivas. Es uno de los componentes fundamentales de las tecnologias web, junto con HTML y CSS.

2. Sintaxis Básica

Javascript es un lenguaje flexible y versátil. A continuación, se presentan algunos elementos básicos de su sintaxis.

Incluir JavaScript en tu página

Puedes incluir JavaScript en tu HTML de varias formas

  • Dentro de una etiqueta <script> en el <head> o al final del <body>:
    • 
<script>
    // Tu código JavaScript aquí
</script>
  • Enlazando un archivo externo:
    • 
<script src="script.js"></script>

3. Variables y Tipos de Datos

Declaración de Variables

En JavaScript, puedes declarar variables usando `var`, `let`o `const`:


var nombre = "Ana";
let edad = 25;
const PI = 3.1416;

Nota:Es recomendable usar `let` y `const` en lugar de `var`para evitar problemas de alcance.

Tipos de Datos

  • String: Cadenas de texto. Ejemplo: `"Hola mundo"`
  • Number: Numeros enteros o decimales. Ejemplo: `42, `3.14
  • BooleanValores lógicos `true` o `false`
  • NullAusencia intencionada de valor
  • UndefinedVariable declarada pero sin valor asignado
  • ObjectColección de propiedades

4. Operadores

Operadores Aritmeticos

Se usan para realizar operaciones matemáticas:

  • `+` Suma
  • `-`Resta
  • `*` Multiplicación
  • `/` División
  • `%`Modulo (resto de la división)

Operadores de Comparación

Permiten comparar valores:

  • `==` Igualdad (compara valor)
  • `===` Estrictamente igual (compara valor y tipo)
  • `!=` Diferente
  • `!==` Estrictamente diferente
  • `>` Mayor que
  • `<` Menor que
  • `>=` Mayor o igual que
  • `<=` Menor o igual que

5. Estructuras de Control

Condicionales


    if (condicion) {
        // código si se cumple la condición
    } else {
        // código si no se cumple la condición
    }

Bucles

For: Se usa cuando conoces el número de iteraciones.


    for (let i = 0; i < 5; i++) {
        console.log(i);
    }

While: Se ejecuta mientras la condición sea verdadera.


    let i = 0;
    while (i < 5) {
        console.log(i);
        i++;
    }

6. Funciones

Declaración de Funciones


    function saludar(nombre) {
        return "Hola, " + nombre + "!";
    }

    let mensaje = saludar("Carlos");
    console.log(mensaje); // "Hola, Carlos!"

Funciones Flecha (Arrow Functions)


    const saludar = (nombre) => {
        return "Hola, " + nombre + "!";
    };

7. Manipulación del DOM

El **DOM (Document Object Model)** es la representación de la página web en un formato que JavaScript puede manipular. Con el DOM, puedes acceder y modificar el contenido, estructura y estilos de tu sitio.

Seleccionar Elementos


    // Por ID
    const titulo = document.getElementById('titulo');

    // Por clase
    const items = document.getElementsByClassName('item');

    // Usando selectores CSS
    const boton = document.querySelector('.boton');

Modificar Contenido


    titulo.textContent = "Nuevo Título";
    titulo.style.color = "blue";

8. Eventos

Los eventos permiten realizar acciones en respuesta a interacciones del usuario, como clics, movimientos del ratón o pulsaciones de teclas.

Escuchando Eventos


    const boton = document.getElementById('boton');

    boton.addEventListener('click', () => {
        alert("¡Has hecho clic en el botón!");
    });

Ejemplo Práctico

Agrega el siguiente código a tu HTML:


    <button id="boton">Haz clic aquí</button>

Y en tu JavaScript:


    const boton = document.getElementById('boton');

    boton.addEventListener('click', () => {
        boton.textContent = "¡Me hiciste clic!";
    });

Ejercicio 1: Calculadora Sencilla

Pista: Usa `prompt()`, funciones y operadores aritméticos.

Desarrolla una aplicación que permita añadir tareas a una lista y marcarlas como completadas al hacer clic.

Pista: Manipula el DOM para añadir elementos y manejar eventos de clic.
